# Resources

This directory contains configuration files used by Databricks assets.

## Files

### `security_genie_instructions.txt`

System instructions for the Security Genie AI assistant. This file defines the assistant's persona, lists the most security-relevant audit log service names, references the available SQL table functions (`failed_authentication_attempts()` and `failed_ip_access_attempts()`), and provides guidance on how to handle time-bounded queries.
