# Apps

Databricks Apps in this repo. Each app has its own README with setup, configuration, and deployment details.

## Apps

| App | Description |
|---|---|
| [`amiga_app/`](amiga_app/README.md) | **Amiga Bricks** — a web-based Commodore Amiga emulator powered by [EmulatorJS](https://emulatorjs.org/) (PUAE core), served via FastAPI, and deployed via Databricks Asset Bundles. Supports single-disk and multi-disk ADF games. Kickstart ROMs and game images are not included (copyrighted). |

See each app's README for prerequisites, local development, and deployment instructions.
