# Kickstart ROM Files

Place your Amiga Kickstart ROM file(s) here.

## Supported files

The app auto-discovers ROM files in this directory by extension (`.rom`,
`.bin`) or by name (any file containing "kick"). Common filenames:

    amiga-os-130.rom
    kick13.rom
    Kickstart v1.3 (A500).rom

A Kickstart 1.3 ROM is recommended for most classic Amiga games. The file
should be exactly 262,144 bytes (256 KB).

If multiple ROMs are present, the first one (alphabetically) is used.

## How to obtain

Kickstart ROMs are copyrighted by Amiga Corporation.
You can legally obtain them from:

- **Amiga Forever** by Cloanto: https://www.amigaforever.com/
  (The Plus Edition includes all Kickstart ROMs as unencoded files)

### Extracting from Amiga Forever (Plus Edition)

After installing on Windows, ROMs are typically found at:

    C:\Users\<you>\Documents\Amiga Files\Shared\rom\

The file `amiga-os-130.rom` is Kickstart 1.3. Copy it into this directory.
